# 🏏 PowerPlay Sports

A full-stack sports e-commerce website built using HTML, CSS, JavaScript, Node.js, Express.js, MongoDB, and Mongoose.

## 🚀 Features

### User Features
- User Registration
- User Login Authentication
- Logout Functionality
- Product Search
- Product Listing from MongoDB
- Add To Cart
- Remove From Cart
- Clear Cart
- Checkout System
- Responsive Design
- Attractive Toast Notifications

### Website Pages
- Home Page
- Products Page
- Offers Page
- Contact Page
- Login Page
- Signup Page
- Cart Page

### Admin/Data Features
- MongoDB Database Integration
- Product API
- User Authentication API
- Product Management using Mongoose

---

## 🛠 Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- bcryptjs
- JSON Web Token (JWT)
